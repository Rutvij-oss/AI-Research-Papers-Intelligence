# 📚 AI Research Papers Intelligence

An AI-powered **Semantic Research Paper Search Engine** that helps users search research papers using Natural Language Processing (NLP). The project performs semantic search, summarizes research papers, extracts keywords, identifies named entities (NER), and supports PDF processing.

---

## Features

- Semantic Search using Sentence Transformers
- Fast Similarity Search with FAISS
- Research Paper Summarization
- Keyword Extraction using KeyBERT
- Named Entity Recognition (NER) using spaCy
- PDF Upload & Text Extraction
- Download Summary
- Search History
- Research Paper Analysis

---

## Installation

Clone the repository

```bash
git clone https://github.com/Rutvij-oss/AI-Research-Papers-Intelligence.git
```

Go inside the folder

```bash
cd AI-Research-Papers-Intelligence
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

Open **Project02.ipynb** in Google Colab or Jupyter Notebook and execute all cells.
