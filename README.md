# AI Research Papers Intelligence

An AI-powered Semantic Research Paper Search Engine built using NLP and Deep Learning.

## Features

- Semantic Search using Sentence Transformers
- FAISS Vector Search
- Research Paper Summarization
- Keyword Extraction using KeyBERT
- Named Entity Recognition (NER)
- PDF Upload and Processing
- Search History
- Download Summary

## Tech Stack

- Python
- Sentence Transformers
- FAISS
- Hugging Face Transformers
- spaCy
- KeyBERT
- PyMuPDF
- Pandas
- Google Colab

## Installation

```bash
pip install -r requirements.txt
```

## Run

Open `Project02.ipynb` in Google Colab or Jupyter Notebook and run all cells.
